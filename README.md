Web-приложение "Объявления о продаже машин".
----

#### Описание проекта:
- Основная страница со всеми объявлениями о продаже машин.
- Страница создания объявления.
- Страница авторизации.

#### Используется:
* Java 8;
* Hibernate;
* PostgreSQL, HSQLDB;
* Servlets, Tomcat;
* Log4j, JUnit, Apache FileUpload;
* Jackson JSON API;
* HTML, CSS, JavaScript, jQuery;
* AJAX.
